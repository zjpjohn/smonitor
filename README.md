# Smonitor
##简介：
一款超简单的主机、系统监控程序<br>
####特点：
无依赖、快速部署

##部署使用：
####需要：
######maven环境（配置模块、打包）
######java环境
######J2EE容器（如：tomcat、weblogic）
####部署方法:
在打包所有模块以后，monitor模块打包会产生一个war包，直接放在容器里运行就可以<br>
本项目为 maven分模块的项目，有几个模块甚至又分了二级模块<br>
每个项目是做什么的，都写在了每个模块的 pom文件中<br>

##二次开发：
#####文件编码:
UTF-8无BOM

#####IDE:
由于使用了三级模块分类，建议使用idea
